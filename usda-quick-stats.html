About NASS

Key resources, free API access, and the release schedule for USDA National Agricultural Statistics Service data.

Using Quick Stats

  • Pick Commodity (corn, soybeans, wheat…)
  • Choose Category (yield, area, production, price)
  • Set Geography (national, state, county)
  • Set a Year range and get data
  • Export as CSV, Excel, or JSON

Free NASS API

  • Free API at quickstats.nass.usda.gov/api
  • Free API key — register at NASS
  • JSON, CSV, and XML output
  • Same parameters as the query builder
  • Built for automated pulls and scripts

Key Data Releases

  • Crop Production Annual — mid-January (final)
  • Crop Progress — Mondays, Apr–Nov
  • WASDE — monthly (see calendar)
  • Prospective Plantings — late March
  • Acreage — late June · Grain Stocks — quarterly
